Conax Bible Assistant Service API

Content Service API Paths

Path /
Example http://localhost:3000/
Description Displays help page.
Path /api/contents/bible
Example http://localhost:3000/api/contents/bible
Description Get the whole Bible content.
Path /api/contents/booknames
Example http://localhost:3000/api/contents/booknames
Description Get the names of the 66 books together with the chapter numbers for each book.
Path /api/contents/{Book Name}
Example http://localhost:3000/api/contents/創世紀
Description Get the whole book (1 of the 66 books).
Path /api/contents/{Book Name}/chapternumbers
Example http://localhost:3000/api/contents/創世紀/chapternumbers
Description Get the chapter numbers of a book.
Path /api/contents/{Book Name}/{Chapter No}
Example http://localhost:3000/api/contents/創世紀/1
Description Get the whole chapter from a book.
Path /api/contents/{Book Name}/{Chapter No}/{Verse No}
Example http://localhost:3000/api/contents/創世紀/1/1
Description Get a verse.
Path /api/contents/{Book Name}/{Chapter No}/{start Verse No}/{End Verse No}
Example http://localhost:3000/api/contents/創世紀/1/1/3
Description Get a range of verses from a chapter.

Search Service API Paths

Path /api/search/bible/{phrase}
Example http://localhost:3000/api/search/bible/耶和華
Description Search in the whole Bible.
Path /api/search/book/{Book Name}/{phrase}
Example http://localhost:3000/api/search/book/出埃及記/耶和華
Description Search all chapters in a specific book.
Path /api/search/chapter/{Book Name}/{Chapter No}/{phrase}
Example http://localhost:3000/api/search/chapter/出埃及記/3/耶和華
Description Search in a specific chapter.
